Discussion

The basis for all audio processing objects in the TTBlue API. TTBlue is licensed under the terms of the GNU LGPL.


#defines

TT_MAX_NUM_CHANNELS

Defines a maximum number of channels that TTBlue objects can reasonably expect to use.

Deprecated:

in version 0.3

#define TT_MAX_NUM_CHANNELS 32
Discussion

This is a hold-over from some older implementations that allocate arrays statically for processing multiple channels of audio.

It is included at the moment for backwards compatibility in the code. It should be respected by hosts of TTBlue objects, but its use in creating new TTBlue objects is heavily discouraged.
